# Алгоритм параллельной очистки шума разночастотных биоинспирированных сигналов

## Описание проекта

Данная работа посвящена разработке универсального алгоритма обработки
и классификации ЭЭГ сигналов применительно к задачам Brain-Computer
Interface (BCI). Алгоритм протестирован на двух принципиально разных
датасетах с разными параметрами записи, что подтверждает его
универсальность для разночастотных биоинспирированных сигналов.

Практическая цель работы — создание системы управления бионическим
протезом на основе сигналов ЭЭГ без использования мышц, что особенно
важно для людей с двигательными нарушениями.

---
## Универсальный алгоритм обработки

Оба эксперимента используют единый пайплайн предобработки и извлечения признаков:

```

Сырые ЭЭГ данные (разные частоты и количество каналов)
↓
Ресэмплинг до 256 Гц
↓
Полосовой фильтр 1-40 Гц
↓
ICA очистка от артефактов (моргание, мышцы, глаза)
↓
Извлечение признаков: FFT + ERSP + ICA компоненты
↓
Ранжирование признаков по F-score (SelectKBest)
↓
Классификация: Random Forest / SVM / Neural Network
```

## Эксперимент 1 — Классификация букв

Подробнее: [experiment_1_letters/README.md](experiment_1_letters/README.md)

Задача — определить по ЭЭГ сигналу какую букву видел испытуемый.
46 файлов по одному на каждую сессию записи. Эпохи усредняются по классам —

## Эксперимент 2 — Классификация движений руки

Подробнее: [experiment_2_hand/README.md](experiment_2_hand/README.md)

Испытуемый выполнял три типа движений ладонью в течение 50 циклов.
Задача — определить по ЭЭГ какое движение совершает человек для
управления бионическим протезом. Каждая эпоха используется как
отдельный пример без усреднения.

### Классы движений

| Класс | Описание | Эпох |
|---|---|---|
| Extension | Поднятие ладони вверх ~60° | 49 |
| Flexion | Опускание ладони вниз ~60° | 49 |
| Palm Forward | Нейтральное положение | 98 |

### Иерархическая система классификации

```
ЭЭГ сигнал
↓
Уровень 1: Покой vs Движение (SVM) — точность 97.5%
↓
Покой    → команда СТОП
Движение ↓
Уровень 2: Extension vs Flexion (SVM) — точность 90%
↓
Extension → команда ВВЕРХ
Flexion   → команда ВНИЗ
```

## Почему точность разная?

Моторные сигналы значительно лучше поддаются классификации:

| Причина | Буквы | Рука |
|---|---|---|
| Примеров на класс | 46 | 65 |
| Физиология | Одна зрительная кора для всех букв | Разные области моторной коры |
| Сила сигнала | Слабые зрительные потенциалы | Сильные моторные потенциалы |
| Различимость классов | Тонкие различия между буквами | Чёткие различия между движениями |
| Случайное угадывание | 10% (10 классов) | 33% (3 класса) |

## Выводы

1. Универсальный алгоритм успешно работает на двух разных датасетах
   с разным количеством каналов (128 и 24) и разными типами сигналов.

2. Для зрительных стимулов наиболее информативны ERSP признаки —
   изменение мощности спектра в затылочных областях.

3. Для моторных сигналов наиболее информативны ICA компоненты —
   99% топ признаков относятся к ICA.

4. Иерархическая система классификации движений руки (~87.8%)
   превосходит результаты аналогичных работ из литературы (53%).

5. Система готова к интеграции с бионическим протезом —
   три чётких команды с точностью 90-97%.
