# Классификация зрительных стимулов по сигналам ЭЭГ

## Описание проекта

Данная работа посвящена решению задачи классификации зрительных стимулов 
по сигналам электроэнцефалографии (ЭЭГ). Испытуемым последовательно 
показывали 10 различных букв, во время просмотра которых записывалась 
электрическая активность мозга с помощью 128-канального ЭЭГ. Цель работы — 
разработать систему, способную по сигналу ЭЭГ определить, какую именно 
букву видел испытуемый в каждый момент времени.

Задача относится к классу задач Brain-Computer Interface (BCI) — 
интерфейс мозг-компьютер — и представляет практический интерес для 
создания систем управления без использования мышц, что особенно важно 
для людей с двигательными нарушениями.

## Датасет

- Формат файлов: `.set` (формат EEGLAB, MATLAB)
- Количество файлов: 46
- Количество электродов: 128 (124 ЭЭГ + 4 глазных EOG)
- Частота дискретизации: 2048 Гц
- Длительность эпохи: 3 секунды (от -1 до +2 сек относительно стимула)
- Количество классов: 10 (буквы s1-s10)
- Количество эпох на файл: от 229 до 310

## Методология

### Шаг 1 — Предобработка сигнала

Каждый файл проходит следующую цепочку обработки:

**Ресэмплинг** — частота дискретизации понижается с 2048 Гц до 256 Гц. 
Это возможно без потери информации, так как нас интересуют частоты до 40 Гц, 
а по теореме Найквиста достаточно 80 измерений в секунду. Снижение частоты 
уменьшает объём данных в 8 раз — с 1.77 ГБ до 226 МБ на файл.

**Фильтрация** — применяется полосовой фильтр 1-40 Гц. Частоты ниже 1 Гц 
содержат медленные дрейфы сигнала не связанные с мозговой активностью, 
частоты выше 40 Гц — преимущественно мышечные артефакты. Все значимые 
мозговые ритмы (тета 4-8 Гц, альфа 8-13 Гц, бета 13-30 Гц, гамма 30-40 Гц) 
сохраняются.

**ICA очистка** — метод независимых компонент (Independent Component Analysis) 
разделяет смешанный сигнал на независимые источники. Первые три компонента, 
соответствующие артефактам моргания и движения глаз, удаляются. Это 
стандартная процедура в обработке ЭЭГ, описанная в работе Makeig et al. (2004) 
"Mining event-related brain dynamics".

### Шаг 2 — Извлечение признаков по классам

После предобработки эпохи группируются по классам (s1-s10) и усредняются 
внутри каждого класса. Это позволяет получить стабильный представитель 
каждого класса для каждого файла. Итого: 46 файлов × 10 классов = 460 примеров.

Для каждого класса извлекаются три типа признаков:

**FFT (Fast Fourier Transform)** — быстрое преобразование Фурье разлагает 
сигнал на составляющие частоты. На выходе получается мощность каждой частоты 
от 1 до 40 Гц для каждого из 128 электродов. FFT показывает из каких частот 
состоит сигнал в среднем, без учёта временной динамики.
Форма признаков: (128 электродов, 115 частотных точек)

**ERSP (Event-Related Spectral Perturbation)** — изменение мощности спектра 
относительно базовой линии (период -0.5 до 0 сек до стимула). Вычисляется 
с помощью вейвлетного преобразования Морле. Положительные значения (ERS) 
означают усиление мощности, отрицательные (ERD) — снижение. ERSP показывает 
как мозг реагирует на стимул в каждом частотном диапазоне. Для экономии 
памяти усреднён по временной оси.
Форма признаков: (128 электродов, 38 частот)

**ICA компоненты** — активации независимых компонентов во времени после 
удаления артефактных компонентов. Каждый компонент представляет отдельный 
физиологически осмысленный источник мозговой активности.
Форма признаков: (20 компонентов, 768 точек времени)

### Шаг 3 — Ранжирование признаков

Все три типа признаков объединяются в единый вектор из 35300 признаков. 
Затем применяется метод SelectKBest с критерием F-score для отбора 500 
наиболее информативных признаков. F-score измеряет насколько сильно каждый 
признак различается между классами.

Результат ранжирования показал, что наибольшую важность имеют признаки 
типа ERSP, что подтверждает теорию — зрительная обработка букв наиболее 
отчётливо проявляется в изменении спектральной мощности затылочных 
электродов.

### Шаг 4 — Обучение классификаторов

Данные делятся на обучающую (80%, 368 примеров) и тестовую (20%, 92 примера) 
выборки. Для подбора оптимальных гиперпараметров используется GridSearchCV 
с 5-кратной стратифицированной кросс-валидацией, обучаемый исключительно 
на обучающей выборке во избежание утечки данных.

Выбраны три классификатора, представляющих разные подходы к классификации:

**Random Forest** — ансамблевый метод на основе деревьев решений. Выбран 
за устойчивость к шуму, что важно для зашумлённых ЭЭГ сигналов, и за 
способность работать с большим количеством признаков без предварительного 
отбора.

**SVM (Support Vector Machine)** — метод опорных векторов. Выбран как 
исторически один из наиболее распространённых методов классификации ЭЭГ 
в научной литературе, эффективен при малом количестве обучающих примеров.

**Neural Network (MLP)** — многослойный персептрон. Выбран как современный 
метод способный улавливать сложные нелинейные зависимости между признаками.

## Результаты

### Точность классификации

| Классификатор | Точность (тест) | Кросс-валидация |
|---|---|---|
| Random Forest | 25.00% | 23.48% ± 3.12% |
| SVM | 31.52% | 38.04% ± 4.51% |
| Neural Network | 34.78% | 39.35% ± 4.20% |

Случайное угадывание для 10 классов составляет 10%. Все три классификатора 
превышают этот порог в 2.5-3.5 раза, что подтверждает наличие различимых 
паттернов в ЭЭГ сигналах для разных зрительных стимулов.

Лучший результат показала нейронная сеть — 34.78% на тестовой выборке. 
Наиболее стабильным оказался SVM с наименьшим разбросом кросс-валидации.

### Анализ по классам (Neural Network)

Лучше всего классифицируется класс **s3** с F1-score = 0.60 — его мозговой 
паттерн наиболее уникален и отличим от остальных классов.

Хуже всего классифицируются классы **s1 и s10** с F1-score около 0.19-0.21 — 
их паттерны активности мозга схожи с другими классами.

### Визуализация результатов

**Средние волны ЭЭГ по классам** — показывают усреднённый потенциал (ERP) 
для каждой буквы на затылочном электроде O1. Классы s3, s4 и s8 демонстрируют 
наиболее выраженные реакции сразу после стимула (0-0.5 сек), что соответствует 
их лучшей классифицируемости.

**ERSP карты** — тепловые карты изменения мощности спектра во времени. 
Во всех классах наблюдается характерная альфа-десинхронизация (ERD) в 
диапазоне 8-13 Гц после стимула — классический признак зрительного восприятия.

**FFT спектры** — частотные спектры по классам практически совпадают, 
что подтверждает низкую различимость классов по FFT признакам в сравнении 
с ERSP.

**Топографические карты** — пространственное распределение активности по 
голове для каждого класса в окне 0-0.5 сек после стимула. Разные классы 
показывают различные паттерны активации, особенно заметные в затылочных 
и центральных областях.

## Выводы

1. Задача классификации 10 зрительных стимулов по ЭЭГ является сложной — 
   лучший результат составил 34.78%, что тем не менее в 3.5 раза превышает 
   случайное угадывание.

2. Наиболее информативными оказались признаки типа ERSP — изменение 
   мощности спектра в затылочных областях, что согласуется с анатомией 
   зрительной коры.

3. Нейронная сеть показала лучший результат среди трёх классификаторов, 
   что объясняется её способностью улавливать нелинейные зависимости 
   между признаками.

4. Основное ограничение работы — малое количество данных (460 примеров 
   для 10 классов), что затрудняет обучение классификаторов.
