# Эксперимент 2 — Классификация движений руки по ЭЭГ

[Вернуться к главной странице](../README.md)

## Описание

Испытуемый выполнял три типа движений ладонью в течение 50 циклов.
Задача — определить по ЭЭГ сигналу какое движение совершает человек
для управления бионическим протезом.

---

## Параметры датасета

| Параметр | Значение |
|---|---|
| Оборудование | ANT Neuro eego EE-511 |
| Формат | BrainVision (.eeg/.vhdr/.vmrk) + CSV |
| Электродов | 24 ЭЭГ |
| Частота дискретизации | 2048 Гц |
| Длительность эпохи | 2 сек |
| Количество классов | 3 |
| Количество примеров | 196 |
| Примеров на класс | 65 |

---

## Классы движений

| Класс | Описание | Эпох | Команда роботу |
|---|---|---|---|
| Extension | Поднятие ладони вверх ~60° | 49 | ВВЕРХ |
| Flexion | Опускание ладони вниз ~60° | 49 | ВНИЗ |
| Palm Forward | Нейтральное положение | 98 | СТОП |

---

## Структура одного цикла эксперимента

```
Palm Forward (2 сек) → Пауза → Extension (2 сек) → Пауза
Palm Forward (2 сек) → Пауза → Flexion (2 сек)   → Пауза
```

Всего 50 циклов. Каждое действие длится ровно 2 секунды.

---

## Пайплайн обработки

```
CSV файл с 24 каналами ЭЭГ и метками движений
↓
Нарезка на эпохи по меткам (2 сек каждая)
↓
Создание MNE объекта с монтажом standard_1020
↓
Ресэмплинг: 2048 → 256 Гц
↓
Фильтрация: полосовой фильтр 1-40 Гц
↓
ICA очистка: 15 компонентов, удалены 2, 9, 13, 14
↓
Извлечение признаков FFT + ERSP + ICA (без усреднения эпох!)
↓
Ранжирование: отобраны 200 из 10512 признаков (F-score)
↓
Классификация: прямая (3 класса) и иерархическая (2 уровня)
```

Ключевое отличие от эксперимента 1 — эпохи **не усредняются**,
каждая является отдельным примером для классификатора.

## Извлечённые признаки

| Признак | Форма | Описание |
|---|---|---|
| FFT | (196, 24, 79) | Мощность частот 1-40 Гц |
| ERSP | (196, 24, 39) | Изменение мощности (усреднено по времени) |
| ICA | (196, 15, 512) | Активации компонентов во времени |
| Итого до отбора | 10512 признаков | Объединённый вектор |
| После отбора | 200 признаков | Наиболее информативные по F-score |

99% наиболее важных признаков — **ICA компоненты**. Моторные сигналы
наиболее информативны именно в пространстве независимых компонентов.

## Выводы

1. Иерархическая система (~87.8%) значительно превосходит
   прямую классификацию 3 классов (75%).

2. ICA компоненты — наиболее информативные признаки для
   моторных задач (99% топ признаков).

3. Palm Forward распознаётся почти идеально (97.5%) —
   нейтральное положение имеет уникальный ЭЭГ паттерн.

4. Система готова к интеграции с бионическим протезом.
