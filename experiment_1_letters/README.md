# Эксперимент 1 — Классификация букв по ЭЭГ

[Вернуться к главной странице](../README.md)

## Описание

Испытуемым последовательно показывали 10 различных букв.
Во время просмотра записывалась электрическая активность мозга
с помощью 128-канального ЭЭГ. Задача — определить по сигналу ЭЭГ
какую именно букву видел испытуемый.

---

## Параметры датасета

| Параметр | Значение |
|---|---|
| Формат файлов | .set (EEGLAB) |
| Количество файлов | 46 |
| Электродов | 128 (124 ЭЭГ + 4 EOG) |
| Частота дискретизации | 2048 Гц |
| Длительность эпохи | 3 сек (-1 до +2 сек относительно стимула) |
| Количество классов | 10 (буквы s1-s10) |
| Количество примеров | 460 (46 файлов × 10 классов) |
| Примеров на класс | 46 |

---

## Пайплайн обработки

```
46 файлов .set
↓
Ресэмплинг: 2048 → 256 Гц (данные уменьшаются с 1.77 ГБ до 226 МБ)
↓
Фильтрация: полосовой фильтр 1-40 Гц
↓
ICA очистка: 20 компонентов, удалены 0, 1, 2 (моргание, движение глаз)
↓
Группировка эпох по классам (s1-s10)
↓
Усреднение эпох внутри каждого класса
↓
Извлечение признаков FFT + ERSP + ICA
↓
Ранжирование: отобраны 500 из 35300 признаков (F-score)
↓
Обучение классификаторов с GridSearchCV
```

---

## Извлечённые признаки

| Признак | Форма | Описание |
|---|---|---|
| FFT | (128, 115) | Мощность частот 1-40 Гц на каждый электрод |
| ERSP | (128, 38) | Изменение мощности спектра (усреднено по времени) |
| ICA | (20, 768) | Активации независимых компонентов |
| Итого до отбора | 35300 признаков | Объединённый вектор |
| После отбора | 500 признаков | Наиболее информативные по F-score |

Наибольшую важность показали признаки типа **ERSP** — изменение
спектральной мощности в затылочных областях. Это согласуется с тем
что зрительная кора расположена именно в затылке.

## Выводы

1. Задача классификации 10 зрительных стимулов является сложной —
   лучший результат 34.78% в 3.5 раза превышает случайное угадывание.

2. ERSP признаки наиболее информативны для зрительной задачи.

3. Основное ограничение — малое количество примеров на класс (46).

4. Для улучшения результатов рекомендуется применение CNN на сырых данных.
